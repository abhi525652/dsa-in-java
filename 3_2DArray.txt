🚀 2D Array Challenge (30 Questions)

This roadmap goes from Beginner → Intermediate → Interview Level.

🟢 Level 1 – Basics (1–10)

Goal: Understand how a matrix works.

 1. Print all elements of a 2D array
 2. Find the number of rows and columns
 3. Sum of all elements
 4. Find the largest element
 5. Find the smallest element
 6. Count even and odd numbers
 7. Search for a target element
 8. Print row-wise sum
 9. Print column-wise sum
 10. Calculate the average of all elements
🟡 Level 2 – Matrix Operations (11–20)

Goal: Learn row and column manipulation.

 11. Print the main diagonal
 12. Print the secondary diagonal
 13. Sum of the main diagonal
 14. Sum of the secondary diagonal
 15. Transpose of a matrix
 16. Check if a matrix is symmetric
 17. Add two matrices
 18. Subtract two matrices
 19. Multiply two matrices
 20. Swap two rows of a matrix
🔵 Level 3 – Pattern & Interview Questions (21–30)

Goal: Placement-level problems.

 21. Reverse each row
 22. Reverse each column
 23. Rotate matrix by 90°
 24. Print matrix in spiral order
 25. Print boundary elements
 26. Find the row with the maximum sum
 27. Find the column with the maximum sum
 28. Search in a sorted matrix
 29. Check if two matrices are equal
 30. Print matrix in zigzag order
